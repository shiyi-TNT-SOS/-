# AI带货视频故事板生成工具

一个面向短视频带货创作的故事板生成工具，用于把产品信息、对标视频拆解、人货场设定、镜头动作和 Seedance 2.0 提示词整合成一套可执行的短视频生成方案。

## 核心功能

### 1. 产品信息填写

输入主推广产品名称后，可使用 AI 一键填充：

- 核心卖点
- 目标人群
- 主讲人设定
- 推荐拍摄场景
- 视频时长
- 拍摄风格

适合快速从一个产品名扩展出完整的短视频创作基础信息。

### 2. 对标视频拆解

支持填写对标视频的拆解维度，包括：

- 前 3 秒如何抓人
- 中间段如何展示产品
- 镜头如何切换
- 人物如何动作
- 结尾如何收口
- BGM 节奏

也支持添加对标视频备注，用于记录账号爆款逻辑、开场痛点、节奏特点等信息。

### 3. 视频上传与链接解析

工具支持添加：

- MP4 视频文件
- MOV 视频文件
- 抖音视频链接
- 小红书视频链接
- 视频号链接
- 快手视频链接

上传本地视频后可读取基础视频信息。  
视频平台链接解析需要配置第三方视频解析 API。

### 4. 六维锁定

通过六个维度锁定视频生成的一致性：

- 人物
- 产品
- 场景
- 口播
- 镜头
- 动作

用于减少 AI 视频生成时人物变形、产品不一致、场景跳变、镜头混乱等问题。

### 5. 故事板生成

根据产品信息和对标拆解结果，生成适合短视频带货的故事板结构。

适合用于：

- Image2 故事板
- Seedance 2.0 视频生成
- 带货短视频脚本规划
- 对标爆款视频复刻
- 产品种草视频设计

### 6. Seedance 2.0 提示词输出

工具会根据前面的设定，整理出更适合 Seedance 2.0 使用的提示词，帮助保持：

- 人物一致
- 产品一致
- 场景一致
- 运镜清晰
- 动作明确
- 节奏稳定
